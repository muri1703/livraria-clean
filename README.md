# Livraria---Eng-de-software

## Descrição
Este projeto implementa um sistema de filtro de livros utilizando Test-Driven Development (TDD).

## Objetivo
Retornar uma lista de livros filtrada por critérios fornecidos pelo usuário.

## Funcionalidades
- Filtrar por gênero
- Filtrar por preço máximo
- Combinar múltiplos critérios

## TDD aplicado
O desenvolvimento seguiu os passos:

1. Escrita dos testes automatizados
2. Execução dos testes (falha inicial)
3. Implementação do código mínimo
4. Refatoração

## Como executar

```bash
python -m unittest test_livro_service.py
